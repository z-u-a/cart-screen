# cart-screen
Basic cart screen using Redux and Hooks
 
Steps: 
1) Clone the repo
2) Switch to develop branch
3) Open terminal
4) Navigate to repo's home directory
5) Run `npm install` followed by `npm start` 
6) Visit http://localhost:3000 in your browser
